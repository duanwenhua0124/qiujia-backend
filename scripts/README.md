# 数据库初始化脚本 - 球家宝贝成长社

## 使用方法

```bash
cd 打卡APP/backend
npm install
npm run init
```

## 验证

执行成功后可以看到任务列表和账号信息。

## 注意事项

- 如需重新初始化，先清空数据: `db.tasks.drop()` 和 `db.users.drop()`
- 管理员账号如已存在不会重复创建
